grabcartoons v2.5

Grab your fix of the funnies.

GrabCartoons is a comic-summarizing utility. It is modular, and it is very easy to write modules for new comics.

Contributions

If you write any new modules, please share them with us! You can post them at the issue tracker, and we will add them to a future release. Or just fork the code in github and then send a pull request.
